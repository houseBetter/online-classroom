  • 什么是微信小程序开发NEXT学位？

    微信小程序开发NEXT学位是淘淘官方自研，由淘淘高级工程师与微信团队共同打造的学习认证项目。课程导师、专家和 Code Reviewer 都来自 BAT 等一线互联网公司。

    上课的形式是怎样的？

    上课采用录播和直播结合的方式，课后还有Code Review、1V1技术辅导、工程师文化直播和同步学习小组等多种学习补充，你可以根据自己的时间安排学习进度，并可以无限制地访问录播视频与自己完成的实战项目。

  • 我可以从此学位中获得什么？

    毕业后，你将获得NEXT学位颁发的小程序开发能力认证，证明你的技术水平，成为小程序开发领域领先的开发者。

    课程是适合小白学习还是需要有一定经验的人学习？

    基础章节着重于对开发工具的介绍，开发语言的讲解，包括WXML、WXSS、WXS脚本等，以及基础组件和API的使用方法。具备一些Html、JavaScript 和 css 基础会更易于上手。进阶章节对一些使用技巧、编程方法、优化手段展开讲解，目的在于提高已具备小程序开发能力的学员，优化对复杂项目的代码组织、团队协作上的管理能力。

  • 淘淘NEXT学位还有其他课程吗？

    目前淘淘 NEXT 学位已经推出以前端、小程序、Node.js 为主题的系列课程，并且还有更多主题课程正在制作中。想要了解更多 NEXT 学位课程，可以查看以下链接：https://ke.qq.com/course/list?mt=1001&st=2004&tt=3322

    如果课程不合适，可以退款吗？

    开始学习课程即视为开课。如果在报名后发现课程内容不适合，可以在付款后的7天内申请退款，将按未学内容比例获得退款；退款将于申请后15-20个工作日内返还原支付账号。以上退款操作不支持使用分期付款的课程。
